# Oven Temperature Quick Reference

\rowcolors{2}{gray!20}{white}

| Use | Temperature |
| --- | ----------- |
| Pizza | 450°F |
| Roasted vegetables / potatoes | 425-450°F |
| Biscuits | 450°F |
| Breads (sourdough, semolina) | 425°F |
| Empanadas | 425°F |
| Cornbread | 400°F |
| Lasagna / casseroles | 375°F |
| Cookies / cakes / muffins | 350°F |
